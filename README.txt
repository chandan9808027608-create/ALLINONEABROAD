FILES INCLUDED:
  index.html      → Homepage
  shop.html       → Shop / Product listing page
  login.html      → Sign In page
  checkout.html   → Checkout page
  contact.html    → Contact page
  style.css       → All styles
  main.js         → All JavaScript (cart, products, interactions)

HOW TO UPLOAD TO HOSTINGER:
1. Log in to hPanel (hpanel.hostinger.com)
2. Go to "File Manager" or use FTP (FileZilla)
3. Navigate to public_html folder
4. Upload ALL files from this folder to public_html/
5. Make sure index.html is in the ROOT of public_html
6. Your site is live at yourdomain.com

CONNECTING A DOMAIN:
- In hPanel → Domains → Add Domain or use your existing one
- Point it to public_html
